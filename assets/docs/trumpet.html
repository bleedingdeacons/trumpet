🎺 Trumpet Plugin

An announcement management plugin for WordPress. Create, schedule, and display announcements on your site with support for maps, related meetings, and automatic expiry.

What Trumpet Does

Trumpet gives you a dedicated Announcements section in your WordPress admin. You can create announcements with rich content, attach a location map, link related meetings, set start and end dates, and display everything on the front end with a simple shortcode.

It's built for organisations that need a simple, reliable way to keep their community informed — and automatically retire outdated announcements.

Requirements

  • WordPress 6.0 or later
  • PHP 8.0 or later
  • Unity plugin — must be installed and activated (Trumpet depends on it for dependency injection, caching, and the meetings system)
  • Advanced Custom Fields (ACF) — used for the announcement editor fields
⚠️
If Unity is not active, you'll see an admin notice: "This plugin requires the Unity plugin to be installed and activated." Trumpet will not load until Unity is running.

Installation

  1. Make sure the Unity plugin is installed and activated.
  2. Upload the trumpet folder to your /wp-content/plugins/ directory, or install via the WordPress plugin uploader.
  3. Go to Plugins → Installed Plugins and activate Trumpet.
  4. You'll see a new Trumpet menu item in the admin sidebar (near the top, below Dashboard).

Creating an Announcement

  1. In the WordPress admin, click Trumpet → Add New Announcement.
  2. Fill in the announcement fields (see Field Reference below).
  3. Click Publish to make it live immediately, or Submit for Review if you need editorial approval.
💡
Announcements use a custom post type, so they are managed separately from regular Posts and Pages. You'll find them only under the Trumpet menu.

Announcement Fields

Each announcement has the following fields, managed through the ACF editor:

General

FieldDescription
TitleThe headline displayed to visitors. Keep it clear and concise.
BodyRich-text content area. Supports paragraphs, bold/italic, links, images, embedded videos, and iframes.
Featured ImageOptional hero image displayed above the body content.
HideToggle to hide the announcement from the public site without deleting it. Useful for drafts or temporary takedowns.

Scheduling

FieldDescription
Start Display DateThe date the announcement becomes visible. If set to a future date, the announcement won't appear on the public site until that date. Leave blank to display immediately.
End DateThe date the announcement expires. After this date it will no longer show on the front end and will appear as "Expired" in the admin list. Leave blank for no expiry.

Dates use the format dd/mm/yyyy (e.g. 25/12/2026).

Location & Map

FieldDescription
Show MapToggle to display an interactive map on the announcement.
LocationIf Show Map is enabled, pick a location using the map picker. Stores latitude, longitude, and address.

Related Meetings

FieldDescription
Related MeetingLink one or more meetings from the Unity meetings system. These appear as clickable links below the announcement body, marked with an icon for online or in-person.

Status Reference

The admin list shows a colour-coded status for each announcement:

StatusMeaning
ActiveVisible on the public site. The end date is in the future (or not set) and the start date has passed.
PendingPublished but the start display date is in the future — not yet visible to visitors.
ReviewSubmitted for editorial review. Not visible until an editor publishes it.
HiddenManually hidden using the "Hide" toggle. Won't display on the front end.
ExpiredThe end date has passed. No longer visible on the public site.
💡
Admin notices appear at the top of the announcements list when there are expired, pending, or awaiting-review announcements, so you can act on them quickly.

Managing Announcements

Editing

Click any announcement title in the admin list to open the editor. Changes are saved when you click Update. Trumpet automatically tracks what changed and clears its cache so visitors see the latest version.

Hiding & Unhiding

To temporarily remove an announcement from the public site without deleting it, open the announcement and toggle the Hide checkbox. The status will change to "Hidden" and it will no longer appear in the shortcode output.

Deleting

From the admin list, hover over an announcement and click Trash. This moves it to the WordPress Trash. To permanently delete it, go to Trash and click Delete Permanently.

Edit Link on the Front End

If you're logged in as an editor or administrator, a small edit icon appears next to each announcement title on the public page. Click it to jump directly to the editor.

Shortcodes

Announcements List

Use this shortcode on any page or post to display all active announcements:

[list_announcements]

This renders each active announcement with its title, featured image, body content, map (if enabled), related meetings, and the "Valid until" date. Announcements that are hidden, expired, or not yet started are automatically excluded.

At the bottom, a footer message directs visitors to email support@aa-bristol.org to submit a new announcement.

Today's Meetings

Display a list of meetings happening today:

[todays_meetings]

This pulls from the Unity meetings system and shows each meeting's time, name (linked to its detail page), and whether it's online or at a physical location. If there are no meetings today, a "No meetings scheduled for today" message is shown.

Today's Meetings Display

The [todays_meetings] shortcode is designed for your homepage or a welcome page. It automatically determines the current day of the week and filters the Unity meeting list accordingly.

Each meeting entry shows:

  • The meeting time
  • The meeting name (as a link)
  • The location — either the physical address, or "Online" for virtual meetings

Admin List View

Navigate to Trumpet → All Announcements to see the full list. The table includes these custom columns:

  • Status — colour-coded badge showing the current state (see Status Reference)
  • Start Date — shows the scheduled start date, or "Started" if the date has already passed
  • End Date — the expiry date

All three columns are sortable — click the column header to sort. This makes it easy to find expired announcements that need updating or pending ones that are coming up.

💡
Quick Edit is intentionally disabled for announcements to prevent accidental changes to the custom fields. Always use the full editor.

Settings

Go to Trumpet → Settings to access plugin configuration. Currently the settings page manages uninstall behaviour:

Data Preservation on Uninstall

This checkbox (enabled by default) controls whether your announcement posts are kept or deleted when you uninstall the plugin:

  • Checked (default) — announcement posts are preserved in the database. You can reinstall later and your data will still be there.
  • Unchecked — all announcement posts and related data are permanently deleted when the plugin is uninstalled.
🚨
If you uncheck "Data Preservation" and then delete the plugin, all announcement data is permanently removed and cannot be recovered. Make sure this is what you want before proceeding.

Uninstalling

Deactivation

When you deactivate Trumpet, the following cleanup occurs automatically:

  • All plugin caches are cleared
  • Scheduled background tasks are removed
  • Custom database tables (logs, meta) are dropped
  • Custom capabilities are removed from roles
  • Plugin options are cleaned up

Your announcement posts are not deleted on deactivation — only on full uninstall (if data preservation is turned off).

Full Uninstall (Deletion)

When you delete the plugin from the WordPress plugins page:

  • If Preserve Data is on (default): only plugin options are removed. Your announcement posts stay.
  • If Preserve Data is off: all announcement posts are permanently deleted alongside plugin options.

Troubleshooting

"This plugin requires the Unity plugin" notice

Trumpet depends on the Unity plugin. Go to Plugins → Installed Plugins, find Unity, and activate it. Once Unity is active, Trumpet will load normally.

Announcements aren't showing on the front end

Check these common causes:

  • The [list_announcements] shortcode is on the page
  • The announcement is published (not in draft or pending review)
  • The Hide toggle is off
  • The Start Display Date has passed (or is blank)
  • The End Date hasn't passed yet

Map not displaying

Ensure Show Map is toggled on and a valid location (with latitude and longitude) has been set. If both lat and lng are 0, the map won't render.

Changes not appearing for visitors

Trumpet uses a 1-hour cache. It clears automatically whenever you save an announcement, but if you have an additional caching plugin (e.g. WP Super Cache, W3 Total Cache), you may need to purge that cache manually.

FAQ

Can I schedule an announcement in advance?

Yes. Set the Start Display Date to a future date. The announcement will appear as "Pending" in the admin and will become visible on the public site once that date arrives.

What happens when an announcement expires?

It is automatically removed from the front-end display. In the admin, it shows an "Expired" status in red. The post itself remains in the database — you can edit the end date to reactivate it or delete it.

Who can create announcements?

Any user with the edit_posts capability (typically Contributors and above) can create new announcements. The settings page requires manage_options (Administrators only).

Can I use Trumpet without the meetings features?

Yes. Simply don't link any related meetings and don't use the [todays_meetings] shortcode. Announcements work independently — meetings are an optional enhancement.

How do I submit a new announcement without admin access?

Email support@aa-bristol.org. This is shown to visitors in the footer of the announcements list.
